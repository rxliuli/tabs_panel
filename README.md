## 场景

Chrome 的标签页太多时几乎没办法查找，所以有非常强烈的需求有一个插件能实现以下功能。

1. 通过快捷键打开一个面板，包含所有当前打开的标签页列表
2. 在列表直接输入即可搜索，同时搜索时标签中的关键字高亮，允许上下移动
3. 标签页的功能允许使用回车切换到选定标签，删除关闭选定标签

> 注：该功能的 idea 来自于 JetBrains 系 IDE 的 Recent Files 功能。
